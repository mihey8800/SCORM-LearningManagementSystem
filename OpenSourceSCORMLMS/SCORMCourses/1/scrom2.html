Стандарты разработки учебных материалов

Система управления обучением
Система управления обучением (СУО) (Instructional Management System – IMS), разработанная Всемирным образовательным консорциумом (Global Learning Consortium, Inc), – это, можно сказать, система управления курсами или учебный сервер или система компьютерной профессиональной подготовки (computer-based training  (CBT) system) или интегрированная обучающая система. СУО является одним из стандартов для учебных серверов, электронных учебных материалов и объединения этих возможностей. СУО – это технические спецификации для разработчиков и авторов курсов по созданию учебных продуктов с возможностью интеграции. Более детальная информация представлена на http://www.imsglobal.org/.

Объектная модель совместно используемого содержания
Объектная модель совместно используемого содержания (ОМСИС) (Sharable Content Object Reference Model – SCORM), разработанная Министерством обороны США (the U..S. Department of Defense) – это набор взаимосвязанных технических стандартов для создания учебных материалов в формате HTML.

Роль ОМСИС
ОМСИС – это технические стандарты, обеспечивающие следующие возможности:

  1. Повторное использование материалов (могут быть изменены и использованы в различных средах).
  2. Функционирование не зависимо от технической и операционной платформы, а так же типа браузера.
  3. Гибкость структуры учебных материалов (нет необходимости вносить изменения при обновлении программного обеспечения).
  4. Доступность для поисковых систем и пользователей.

Особенности ОМСИС

  1. Структура материала:
    • Содержание
    • Учебные объекты
  2. Технические требования:
    • Совместимость
    • Удобная навигация
    • Представление в виде пакета

Учебные объекты – это?
Учебные объекты  – это набор учебных элементов  для совместного доступа (Sharable Content Objects - SCOs) , описанные в  файле содержания (manifest file).     

SCOs …
«Учебный элемент для совместного доступа – это набор ресурсов, который представляет собой отдельную единицу учебных материалов, совместимую с ОМСИС требованиями по времени выполнения."

SCORM комплекты
SCORM комплект (SCORM asset) – это один или больше ресурсов, которые могут быть размещены среди совместно используемых учебных элементов. Комплект должен содержать соответствующие метаданные для обозначения его в SCORM системе хранения.

Средства для разработки стандартизованных учебных материалов

  1. Модуль для Macromedia Dreamweaver MX/MX2004 “Manifest Maker for ADL SC 2.0.0”
  2. courseGenie
  3. Microsoft Producer

Manifest Maker for ADL SC 2.0.0
Manifest Maker обеспечивает быстрый и легкий способ создания файла manifest. Но Manifest Maker имеет ограничение на количество учебных объектов, которые могут быть включены в SCORM-пакет. В таком случае, чтобы добавить дополнительные данные, необходимо отредактировать XML-файлы вручную.Эта версия 2.0 для ADL SCORM 1.2. Обновления Вы можете найти на  http://exchange.macromedia.com

courseGenie - http://www.coursegenie.com/info.htm
CourseGenie – это инструмент, который поможет Вам создавать учебные материалы в MS Word и эффективно преобразовать в динамический онлайн-курс. Из одного документа MS Word Вы сможете сгенерировать ряд web-страниц, включая меню навигации и интерактивные элементы.

Если у Вас уже есть материал такой, как тексты лекций или раздаточные материалы в формате MS Word, Вы можете начать с его  преобразования. Вы можете так же использовать courseGenie, чтобы создать полностью новые материалы.

Возможности courseGenie
Вы можете создавать:

  • Навигация
  • Формирование содержания (оглавления)
  • Форматирование
  • Ссылки
  • Всплывающие окна
  • Включения 
  • Вопросы для самопроверки
  • Windows Media, flash-анимация и QuickTime-видео
  • Потоковое аудио и видео
  • IMS метаданные

О MS Producer
 Microsoft Producer для Microsoft Office PowerPoint 2003 – это текущая версия популярной надстройки для Microsoft PowerPoint, которая помогает Вам легко добавить и опубликовать аудио, видео, слайды и изображения. Это идеальный инструмент для создания интересной презентации или учебных материалов.

Источник - http://www.microsoft.com/office/powerpoint/producer/prodinfo/default.mspx

Возможности MS Producer

  • Разработать содержание тренинга/учебных материалов.
  • Преподаватели, учителя, авторы статей и т.д. могут создать с помощью Producer учебные материалы, которые поддерживают стандарты «Система управления обучением» (СУО) (Instructional Management System – IMS), и «Объектная модель совместно используемого содержания» (ОМСИС) (Sharable Content Object Reference Model – SCORM).
  • Создавать высококачественные онлайн-презентации.
  • Создавать различные аудио/видео дополнения к документам.
  • Внедрять фрагменты потокового аудио и видео и т.д.
